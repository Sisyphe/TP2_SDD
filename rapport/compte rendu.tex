\chapter{Présentation du TP}

\section{Description}

L'objectif de ce TP et de créer une structure de donnée permettant de stocker un dictionnaire sous la forme d'un arbre.

Le programme prend en argument un lien vers un fichier qui contient un arbre sous forme parenthésé.

Ensuite un menu permet d'insérer un mot dans le dictionnaire créé, d'afficher le dictionnaire ou de quitter le programme.

\section{La structure de données}

Les données sont composées de liste chainées organisées en lien vertical lien horizontal. Chaque élément contient :
\begin{itemize}
\item une lettre (char)
\item un pointeur sur son frère
\item un pointeur sur son fils
\end{itemize}

Les algorithmes de création et de gestion des arbres nécessite l'utilisation de files et de piles. C'est pourquoi nous avons créé deux autres structures de données et les algorithmes qui permettent de les utiliser : une structure de gestion de pile et une structure de gestion de file.
